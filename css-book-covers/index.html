CSS book covers

This project was originally intended to be a simple book cover but I'd read about 3D rotations in CSS and I wanted to try something new.

Then I realised that I could apply transitions to 3D transformations, so I did!

I loved making this page with zero Javascript. Play around with the devtools if you want!
